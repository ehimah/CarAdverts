# CarAdverts REST API with ASP.NET Core

## Testing it out

1. Clone this repository
2. Build the solution using Visual Studio, or on the [command line](https://www.microsoft.com/net/core) with `dotnet build`.
3. Run the project. The API will start up on http://localhost:58053, or http://localhost:5000 with `dotnet run`.
4. Use an HTTP client like [Postman](https://www.getpostman.com/) or [Fiddler](https://www.telerik.com/download/fiddler) to `GET http://localhost:58053`.
